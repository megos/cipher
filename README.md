# cipher
Encrypted in various languages


## Java
- Required Java 8 Higher

#### Usage

```
$ javac -encoding UTF8 CipherSample.java
$ java CipherSample
```

#### Result
```
CAX/zDb/Vdu/063a7fE+qQ==
hello
```

## PHP
- Required PHP 5.1.6

#### Usage

```
$ php CipherSample.php
```


#### Result
```
Original Data : hello
Encrypted Data : CAX/zDb/Vdu/063a7fE+qQ==
Decrypted Data : hello
```
